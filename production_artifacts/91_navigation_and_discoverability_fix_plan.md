# 91 Navigation And Discoverability Fix Plan

## Plan Applied

### Creator shell

- Replaced the flat action-row shell with a clearer two-tier workspace structure.
- Promoted core creator flows into primary navigation:
  - Overview
  - Portfolio
  - Bookings
  - Jobs
  - Finance
- Grouped Messages, CRM, and Calendar into a secondary coordination cluster.
- Added a stronger top workspace framing panel so users immediately understand what Museio is organizing for them.

### Public and auth entry

- Reworked the home page to sell the product direction, not just expose a route list.
- Reframed sign-in as entry into a premium creator operating system rather than a basic form screen.

### Mobile-first alignment

- Shifted the mobile shell from a placeholder explanation card to a quick-action model with stronger hierarchy.

## Expected UX Improvement

- faster orientation when entering the workspace
- less hunting for core sections
- clearer distinction between flagship work and secondary tools
- better narrative continuity between public brand surfaces and creator operations

## Risks

- Some deeper creator screens still rely on older page-level structure and will benefit from further bespoke redesign after this phase.
- Discoverability is materially improved, but the creator detail pages for booking/messages/CRM can still be refined further.

## Approval Gate

This phase implemented the structural navigation improvements. The next approval question is whether to continue into deeper page-specific interaction polish or preserve this as the preview-ready shell baseline.
